# 11-ARCHIVE

This folder holds resolved incidents, completed reviews, retired documents, and historical records.

## What goes here

- Resolved incident reports (from `00-INBOX/`)
- Completed postmortems
- Retired SOPs or runbooks (when replaced by newer versions)
- Completed weekly and monthly reviews

## Structure

```
11-ARCHIVE/
├── Incidents/
│   └── Incident — YYYY-MM-DD — [description].md
├── Postmortems/
│   └── Postmortem — YYYY-MM-DD — [description].md
├── Reviews/
│   └── Weekly — Week of YYYY-MM-DD.md
│   └── Monthly — YYYY-MM.md
└── Retired/
    └── [retired documents]
```

## Naming conventions

Incidents: `Incident — YYYY-MM-DD — [brief description].md`
Postmortems: `Postmortem — YYYY-MM-DD — [brief description].md`
Reviews: `Weekly — Week of YYYY-MM-DD.md` / `Monthly — YYYY-MM.md`
