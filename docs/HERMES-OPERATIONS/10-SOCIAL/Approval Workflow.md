# Approval Workflow — Social Content
**Version:** 1.0
**Last Modified:** 2025-06-06

---

## The Rule

No social content is published without Allan Turner's explicit approval. This is not a default — it is the only workflow that exists. There is no alternate path.

---

## Workflow

```
Hermes drafts post
        │
        ▼
Post staged in 10-SOCIAL/Drafts/ with date and type
        │
        ▼
Allan reviews draft
        │
        ├── Approved → Allan or designated publisher posts manually
        │
        └── Rejected / Revision needed → Hermes revises per Allan's direction
```

---

## Draft Staging

When Hermes creates a social draft, it is saved to:

```
10-SOCIAL/Drafts/YYYY-MM-DD — [Post Type].md
```

Each draft file contains:
- Post text
- Content type
- Data source (what game, stat, or article the post references)
- Drafted time

---

## What Allan Reviews

Allan reviews:
- Factual accuracy of the post
- Tone alignment with Barrel Proof voice
- Appropriateness for posting at that moment
- Whether the post adds value to Barrel Proof's presence

Allan may approve as-is, request revisions, or decline to post. All three are valid outcomes.

---

## Timing

Hermes drafts posts as content becomes available (post-game, post-publication). Allan posts at his discretion. Hermes does not set posting deadlines or apply pressure on timing.
