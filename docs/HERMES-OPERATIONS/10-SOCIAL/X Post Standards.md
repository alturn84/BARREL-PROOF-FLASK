# X Post Standards
**Version:** 1.0
**Last Modified:** 2025-06-06

---

## Core Rule

Hermes drafts. Allan approves. Nothing posts without explicit authorization.

This rule has no exceptions. No urgency, no breaking news, no "obvious" post justifies bypassing Allan's approval.

---

## Post Types

| Type | Trigger | Content |
|------|---------|---------|
| Score recap | After game completion | Final score, key stat, 1 sentence |
| Standings update | After standings change significantly | Division leader, key movement |
| Advanced Scout promo | When Advanced Scout is published | Game, matchup angle, link |
| DFS content | When DFS analysis is published | Key call, confidence note, link |
| Press Box promo | When daily content publishes | Lead item, link |

---

## Writing Standards

**Lead with the fact.**
The first word of a post should be substantive — a team name, a number, a player name. Not "Wow" or "Check out" or "Big night."

**Be precise.**
"Yamamoto: 7 IP, 1 ER, 10 K" is better than "Yamamoto was great tonight."

**Be brief.**
Most posts should be under 200 characters. The full character limit is available but rarely needed.

**No manufactured excitement.**
Use exclamation points only when the result genuinely warrants it. A 6–4 win in May does not warrant an exclamation point.

**No hashtag clutter.**
Use hashtags only when they add discoverability value. Maximum two per post. Never use trending hashtags that are unrelated to the content.

---

## Formatting

Score recaps:
```
[Team] [W/L] [Score]
[Key stat or moment — 1 sentence]
[Optional: link]
```

Example:
```
Dodgers 7, Giants 2
Yamamoto: 7 IP, 1 ER, 10 K. LA wins three straight.
```

Promo posts:
```
[Specific angle or finding] →
[Link]
```

---

## What Hermes Does Not Post

- Opinion or takes on player/team controversies
- Anything that could be read as a prediction
- Content that isn't directly tied to Barrel Proof data or coverage
- Replies to other accounts (Hermes drafts only original posts)
