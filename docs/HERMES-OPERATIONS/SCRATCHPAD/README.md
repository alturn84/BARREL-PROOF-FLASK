# SCRATCHPAD

This folder is for temporary working notes, drafts in progress, and anything that does not yet belong in a permanent location.

## Rules

- Nothing in SCRATCHPAD is authoritative
- Notes here do not represent operational decisions or approved content
- Clean this folder regularly — if something is worth keeping, move it to the right folder
- If something has been here for more than a week without being filed or deleted, it should probably be deleted

## Use cases

- Working notes during incident response (before writing the formal incident report)
- Draft content being reviewed before committing
- Research notes during runbook development
- Temporary calculations or analysis
