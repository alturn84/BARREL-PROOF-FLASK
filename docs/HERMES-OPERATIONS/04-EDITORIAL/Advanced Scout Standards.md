# Advanced Scout Standards
**Version:** 1.0
**Last Modified:** 2025-06-06
**Applies To:** Advanced Scout preview content

---

## What Advanced Scout Is

Advanced Scout is Barrel Proof's analytical preview feature. It is the most data-intensive content type on the site and is targeted at readers who follow MLB closely and want genuine analytical depth — not a restatement of the matchup, but a breakdown that reveals something meaningful.

---

## What Advanced Scout Is Not

- It is not a game prediction
- It is not a recap of each team's record
- It is not a rewrite of the Game of the Day preview
- It is not a collection of general observations

---

## Section Standards

### The Pitching Matchup
The pitching section should go beyond ERA. It should cover:
- Recent form (last 3–5 starts) with specific numbers
- Pitch mix and tendencies if available
- What this starter does well and where they are vulnerable
- How the opposing lineup matches up against those tendencies

The section should not:
- Repeat biographical information already obvious from the matchup
- State that a pitcher "has been good" without supporting statistics
- Pad with general comments about pitching being important

### Lineup Advantages
This section is specifically about batter vs. pitcher matchup data. It should:
- Identify specific hitters who have meaningful history against today's starter
- Explain why the edge exists (pitch type, zone coverage, spray chart tendency)
- Acknowledge small sample sizes when relevant

This section should not:
- List every hitter in the lineup
- State that a hitter is "dangerous" without data
- Reference matchup data that is not in the provided dataset

### Bullpen Watch
This section should focus on availability and workload, not general bullpen strength. It should cover:
- Which arms have been used heavily in the last 3 days
- Whether the team's primary leverage arm is available
- Any known availability restrictions

### Key Variable
This is the most valuable section. It should identify one specific factor that most standard previews will not cover. Good examples:
- A defensive alignment change affecting a specific hitter's spray chart
- A pitcher's performance in day games vs. night games
- Weather impact on a specific ballpark dimension
- A managerial tendency in specific leverage situations

The key variable should be genuinely specific, not a generic observation.

---

## Language Standards for Advanced Scout

Use correct baseball terminology:
- ERA, FIP, xFIP, WHIP, OPS, wOBA, wRC+, K%, BB%, BABIP
- High leverage, leverage index
- Run expectancy
- Spray chart, pull tendency, opposite field rate

Do not over-explain basic statistics to an audience that already follows the game. Advanced Scout readers know what ERA means.

---

## When Data Is Missing

If a required data type is unavailable (BVP data missing, bullpen workload data absent), say so explicitly in the section:

Example: *"Batter vs. pitcher historical data was unavailable for this matchup."*

Never estimate or fabricate to fill a section. An acknowledged gap is better than invented analysis.

---

## Quality Check

Before committing Advanced Scout content:
- [ ] All four sections present
- [ ] Every statistic in the Pitching Matchup section verified against source data
- [ ] Lineup Advantages section references specific matchup data, not general reputation
- [ ] Bullpen Watch covers the last 3 days of usage
- [ ] Key Variable is genuinely specific and not a generic observation
- [ ] No predictions of outcome
- [ ] No fabricated statistics or estimates presented as fact
