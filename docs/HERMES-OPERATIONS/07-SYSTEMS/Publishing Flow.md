# Publishing Flow
**Version:** 1.0
**Last Modified:** 2025-06-06

---

## Overview

This document maps the complete path from data collection to live publication for Barrel Proof's daily content.

---

## Full Publishing Flow

```
MLB Data API
    │
    ▼
Cron Job 1 — 6:00 AM ET
    ├── Fetch game data
    ├── Parse and validate
    ├── Generate game cards
    └── Write JSON to /workspace/barrel-proof/
            │
            ▼
Cron Job 2 — 6:15 AM ET
    ├── Read JSON input
    ├── Send to Gemini API with prompts
    ├── Generate all content types
    ├── Validate content output
    └── Write content to /workspace/barrel-proof/
            │
            ▼
Post-Pipeline
    ├── git add [validated files]
    ├── git commit -m "morning update YYYY-MM-DD — ..."
    ├── git push origin main
    └── Render webhook triggered
            │
            ▼
Render Build
    ├── Clone updated repo
    ├── Run build command
    └── Deploy to production
            │
            ▼
Live Site — target: complete by 6:30 AM ET
    └── Hermes verifies: homepage, content sections, no errors
```

## Content Visibility by Section

| Content Type | Where It Appears | Generated By |
|-------------|-----------------|-------------|
| Game Cards | Homepage / Games section | Cron Job 1 |
| Press Box | Homepage / Editorial section | Cron Job 2 |
| Around the League | Editorial section | Cron Job 2 |
| Game of the Day | Homepage feature | Cron Job 2 |
| Game to Watch | Homepage feature | Cron Job 2 |
| Advanced Scout | Advanced Scout section | Cron Job 2 |
| Game Summaries | Individual game pages | Cron Job 2 |

## Failure Handling in the Flow

| Stage | Failure | Action |
|-------|---------|--------|
| Cron Job 1 | Data not collected | Follow `03-RUNBOOKS/Cron Failure Recovery.md` |
| Cron Job 1 | Validation fails | Follow `03-RUNBOOKS/Data Validation Failure.md` |
| Cron Job 2 | Gemini API fails | Follow `03-RUNBOOKS/Gemini API Quota Failure.md` |
| Cron Job 2 | JSON missing | Follow `03-RUNBOOKS/Missing JSON Recovery.md` |
| Git push | Push fails | Follow `03-RUNBOOKS/Git Conflict Recovery.md` |
| Render | Build fails | Follow `03-RUNBOOKS/Render Deployment Failure.md` |
