# DFS Checklist
**Version:** 1.0
**Last Modified:** 2025-06-06

---

## Pre-Analysis Checklist

Complete before running DFS analysis each day:

### Data Verification
- [ ] Starting pitchers confirmed for today's slate
- [ ] Lineup data is current (check for late scratches)
- [ ] Injury report reviewed — no relevant players missing from analysis
- [ ] Weather data checked for outdoor games
- [ ] Vegas lines and totals available

### Slate Assessment
- [ ] How many games are on the slate? (Full slate vs. small slate — adjust recommendations accordingly)
- [ ] Are there any games with weather concerns?
- [ ] Any significant park factor games (Coors, other offensive parks)?
- [ ] Any high-implied-total games that anchor the slate?

---

## Analysis Checklist

### Pitching
- [ ] 2–3 pitcher targets identified
- [ ] Each pitcher has at least 2 supporting data points
- [ ] Confidence level assigned to each
- [ ] Salary context noted

### Hitting
- [ ] 4–6 hitter targets identified across price tiers
- [ ] At least one value play under median salary
- [ ] Each hitter has specific matchup reasoning (not just "he's good")
- [ ] Batting order position noted for each

### Value Plays
- [ ] 2–3 low-salary upside plays identified
- [ ] Each has a specific reason the salary is undervalued
- [ ] Risk factors acknowledged

### Stacks
- [ ] Potential stack identified (or explicitly noted as absent)
- [ ] Stack reasoning tied to data (implied total, pitcher matchup, park)

---

## Output Checklist

Before submitting DFS analysis:

- [ ] All recommendations include confidence levels
- [ ] All recommendations cite specific supporting data
- [ ] No statistics invented or estimated
- [ ] Slate overview present
- [ ] Analysis framed as analysis, not guarantees
- [ ] No recommendations based solely on name recognition
